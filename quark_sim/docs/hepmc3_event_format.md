# HepMC3 Event Format

This document details the HepMC3 event record format used to store the Monte Carlo DIS events generated by the PYTHIA 8 backend.

## HepMC3 ASCII Format Overview

The event file (`events.hepmc3`) is written in the HepMC3 ASCII format, which is the standard event record serialization for high-energy physics generators. The file structure consists of:

1. **Header Block**:
   Specifies the format version (e.g., `HepMC::Version 3.03.00`) and the units used in the event record.
2. **GenRunInfo Block**:
   Contains global run-level information, such as tool versions, weight names, and configured parameters.
3. **Event Blocks**:
   Each event starts with an Event line (`E`) and is followed by:
   - Attributes (`A`) detailing event-level properties.
   - Vertices (`V`) specifying decay or interaction points in space-time.
   - Particles (`P`) representing the tracks entering or exiting each vertex.

---

## Units and Coordinate System

- **Momentum and Mass**: Measured in **GeV** (Giga-electron-volts).
- **Space and Time**: Coordinates of vertices are measured in **mm** (millimeters) and **mm/c** (millimeters per speed of light).
- **Lepton Direction**: The incoming lepton beam goes in the $+z$ direction.
- **Proton Direction**: The incoming proton beam goes in the $-z$ direction.

---

## Particle Classifications and Status Codes

Particles in the HepMC3 record are assigned status codes that describe their nature:

- **Beam Particles** (Status `4`): The initial state beams (incoming electron and proton).
- **Stable Final-State Particles** (Status `1`): Particles that do not decay within the generator's geometry and can be detected.
- **Decayed/Intermediate Particles** (Status `2`): Particles that decay inside the generator (e.g., unstable resonances like $Z^0$, hadrons like $\pi^0$ or $K^0_S$ that decay).
- **Documentation/Hard Process** (Status `21` - `29`): Particles representing the core hard-scattering subprocess before showers and hadronization.

### Identifying Key Event Components

- **Incoming Beams**: First two particles in the event record with status `4` (electron: ID `11`, proton: ID `2212`).
- **Scattered Lepton**: A stable particle (status `1`) with PDG ID `11` (electron) descended from the incoming electron beam.
- **Hadronic Final State**: All other stable particles (status `1`) carrying baryon number, charge, or neutral energy, resulting from the hadronization of the struck quark and the proton remnants.
- **Proton Remnant**: Formed by the spectator partons from the proton. They carry most of the forward ($-z$) momentum.
